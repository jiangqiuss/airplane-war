# ✈️ 飞机大战 - HTML5 射击游戏

经典街机风格的飞机大战游戏，使用纯 HTML5 + Canvas 实现。

## 🎮 游戏特性

- **玩家飞机**：方向键 / WASD 移动，空格射击
- **多种敌人**：普通、快速、坦克、Boss 四种类型
- **道具系统**：回血 ❤️、加速射击 ⚡、护盾 🛡️
- **粒子特效**：爆炸和击中效果
- **触屏支持**：滑动移动，自动射击
- **最高分记录**：本地存储

## 🚀 开始游戏

直接在浏览器中打开 `index.html` 即可游玩。

或者访问 GitHub Pages 在线体验：[点击这里](https://jiangqiuss.github.io/airplane-war/)

## 🕹️ 操作方式

| 操作 | 按键 |
|------|------|
| 移动 | 方向键 / WASD |
| 射击 | 空格键 |
| 触屏移动 | 滑动屏幕 |
| 触屏射击 | 自动 |
