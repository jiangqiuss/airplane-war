# 🚀 星际掠食者 - Roguelike 飞机大战

一款基于肉鸽（Roguelike）机制的 HTML5 飞机射击游戏，每局随机升级、永久成长、排行榜系统。

## 🎮 在线游玩

👉 [点击这里开始游戏](https://jiangqiuss.github.io/airplane-war/)

## ✨ 游戏特色

### 🎲 肉鸽核心系统
- **23种随机升级**，4个稀有度（普通/优秀/稀有/史诗/传说）
- 每波结束从3选1（幸运≥3时变为4选1）
- 升级可叠加多级，形成不同 Build

### 👾 6种敌人 + Boss
| 敌人 | 特点 |
|------|------|
| 基础战机 | 直线移动 |
| 追踪者 | 锁定玩家追击 |
| 射手 | 向玩家发射子弹 |
| 重型战机 | 高血量坦克 |
| 蜂群机 | 快速蛇形移动 |
| 分裂者 | 击杀后分裂为3个小兵 |
| Boss（每5波） | 4种Boss，3阶段，多种攻击模式 |

### 🛸 6架可解锁战机
| 战机 | 特点 | 被动技能 | 价格 |
|------|------|----------|------|
| 猎鹰号 | 均衡型 | 闪避率5% | 免费 |
| 重锤号 | 重装甲·高伤害 | 受击减伤+3 | 300💰 |
| 幻影号 | 极速·小判定框 | 判定框-30% | 500💰 |
| 雷霆号 | 连射狂魔 | 每次射击+1子弹 | 800💰 |
| 吸血鬼号 | 击杀回血 | 击杀回血+5 | 600💰 |
| 新星号 | 炸弹之王 | 炸弹伤害翻倍 | 1000💰 |

### 💰 永久成长系统
- 6种永久强化（各3-5级）
- 金币通过每局游戏获得
- 所有进度自动存档（localStorage）

### ⚡ 随机事件系统
- 每3波有概率触发随机事件
- 6种事件：补给站、军火库、弹药补给、护盾充能、黄金雨、幸运降临

### 🏆 排行榜系统
- 每局结束自动记录
- 显示波次、击杀、金币、战机、评价等级
- 最多保存20条记录

### 🎯 战斗系统
- **暴击系统**：暴击率+暴击伤害可叠加
- **炸弹清屏**：全屏伤害+清除敌方子弹
- **护盾机制**：独立于血量的额外防御层
- **护卫无人机**：环绕自动射击
- **生命汲取**：攻击回血
- **寒冰弹**：减速敌人
- **荆棘装甲**：反弹伤害
- **狂暴之心**：血量越低伤害越高
- **击杀连击**：COMBO系统
- **闪避系统**：部分战机自带闪避

## 🕹️ 操作方式

| 操作 | 按键 |
|------|------|
| 移动 | 方向键 / WASD |
| 炸弹 | 空格 / E |
| 触屏移动 | 滑动屏幕 |

## 🛠️ 技术栈

- HTML5 Canvas
- 原生 JavaScript
- 无任何外部依赖
- localStorage 本地存储
